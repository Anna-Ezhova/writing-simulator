Deutsch-Prüfungs-Simulator

Diese App ist ein Simulator für den Schreib-Teil der DTZ-Prüfung.
Sie können eine Beispiel-Aufgabe wählen.
Oder Sie schreiben eine eigene Aufgabe.
Dann schreiben Sie Ihren Text.
Die App bewertet Ihren Text automatisch mit Google AI.
So üben Sie für die Prüfung.
So verbessern Sie Ihr Schreiben.

Aufgabe

Sie können eine Beispiel-Aufgabe wählen.
Sie können diese Aufgabe lösen.

  • Aufgabe A: Brief an Vermieter
  • Aufgabe B: E-Mail an Nachbarin

Oder Sie schreiben eine eigene Aufgabe.
So üben Sie für verschiedene Deutsch-Prüfungen.

Bitte schreiben Sie Ihre Aufgabe unten in das Feld.
